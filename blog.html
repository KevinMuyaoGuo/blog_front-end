Maven 详解


1. Maven 介绍 Maven 是 Apache 下的一个纯 Java 开发的开源项目。基于项目对象模型（POM: Project Object Model）概念，Maven 利用一个中央信息片断能管理一个项目的构建、报告和文档等步骤。 Maven 是一个项目管理工具，可以对 Java 项目进行构建、依赖管理。 Maven 详解 功能 Maven 统一集中管理好所有的依赖包，不需要程序员再去寻找。 对应第三方组件用到的共同 jar，Maven 自动解决重复和冲突问题。 Maven 作为一个开放的架构，提供了公共接口，方便同第三方插件集成。程序员可以将自己需要的插件，动态地集成到 Maven，从而扩展新的管理功能。 Maven 可以统一每个项目的构建过程，实现不同项目的兼容性管理。 Maven 主要功能以及概念介绍 教程 Maven 教程 推荐书籍：《Maven 实战》 官方文档 Maven 官方文档
